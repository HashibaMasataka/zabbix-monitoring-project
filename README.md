# 文化祭ネットワーク・サーバ監視基盤構築プロジェクト (Zabbix)

文化祭の安定したインフラ運用を目指し、Proxmox上の仮想環境にZabbixを用いたリアルタイム監視基盤を構築しました。Wi-Fiアクセスポイントの稼働状況や、各ホストの生存確認を自動化・可視化しています。

## 🛠 構築環境・使用技術
- **監視ツール:** Zabbix
- **仮想化プラットフォーム:** Proxmox VE
- **監視対象:** Wi-Fiアクセスポイント、サーバ各種
- **プロトコル・技術:** SNMP, Ping (ICMP), HTTP Agent, JavaScript (データ前処理)

## 🚀 実装内容・特徴
- **自動インフラ監視:** Pingによるホストの生存確認およびSNMPを用いたネットワーク機器のステータス取得。
- **実用的なダッシュボード:** 障害発生時に即座に気づけるよう、リアルタイムで可視化されたダッシュボードの構築。
- **柔軟なアラート設定:** トリガー条件を細かくチューニングし、必要な異常検知を自動化。

## 💡 トラブルシューティング & 工夫した点
構築にあたり、発生した以下の技術的課題を自力で調査・解決しました。
1. **SNMP MIBファイルの不整合解決:** ベンダー特有のMIBファイルの配置と設定を最適化し、正確な機器情報の取得に成功。
2. **データベース＆権限エラーの解消:** Zabbix初期構築時のDB接続エラーやパーミッション問題をログから特定し修正。
3. **HTTP AgentとJavaScriptによるデータ整形:** 標準機能だけでは取得が難しいデータを、HTTP Agentでのデータ収集 ＋ JavaScriptによる前処理（Preprocessing）を組み合わせて自動化。
